# Universal Aura Suite

A premium, high-fidelity command workstation containing over 19+ client-side developers, design, and content utilities. Engineered for speed, atomic layout precision, and offline accessibility.

## 🚀 Features

* **AI Copywriting & Code Suite:** Fully context-aware writing partners, text simplifiers, translators, and semantic code explainers powered by the Google Gemini model.
* **Pixel-Perfect Media Engines:** High-quality image compression, precise resizing, interactive color picker, and layout palette extractors.
* **Strict Developer Utilities:** JSON Formatter, Base64 encoder/decoder, Hash generation, Regex helper, and JWT inspectors.
* **Universal Math & Formatting:** Advanced age calculators, universal unit convertors, percentages, and case converters.
* **Persistent Command Center:** Beautiful, fully secure bento-grid dashboard tracking usage statistics, favorite tools, and detailed activity logs.

---

## 🎨 Sleek Design System

The layout is modeled after a premium, high-contrast digital workspace:
* **The Obsidian Theme:** Deep off-blacks (`#030303`), smooth semi-transparent glass cards (`bg-white/5`), thin border highlights (`border-white/5`), and subtle radial glows around hover triggers.
* **Adaptive Typography:** Headings utilize **Space Grotesk** for clean, modern geometric curves, while status metrics use **JetBrains Mono** for low-friction reading.
* **Micro-Animations:** Interactive components scale up subtly (`hover:scale-[1.01]`) and transition color values smoothly over `300ms`.

## 🎹 Keyboard Shortcuts

Accelerate your workflow with responsive workstation-wide keyboard controls:

* <kbd>Ctrl</kbd> + <kbd>K</kbd> / <kbd>Cmd</kbd> + <kbd>K</kbd> — Toggle Command Palette (Fuzzy Search)
* <kbd>Ctrl</kbd> + <kbd>/</kbd> / <kbd>Cmd</kbd> + <kbd>/</kbd> — Open Command Palette & Focus Search Input
* <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd> — Instantly Toggle Theme (Dark / Light Mode)
* <kbd>Esc</kbd> — Close active palette or open dialogs

## 🛠️ Local Development

1. **Clone & Install Dependencies:**
   ```bash
   npm install
   ```

2. **Configure Environment Variables:**
   Create a `.env` file at the root or define variables in your development console:
   ```env
   GEMINI_API_KEY=your-gemini-api-key-here
   ```

3. **Run Dev Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the workstation in your browser.
