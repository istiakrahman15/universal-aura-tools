# Changelog — Universal Aura Suite

All notable changes to the Universal Aura Suite will be documented in this file. This project adheres to Semantic Versioning (`vMAJOR.MINOR.PATCH`).

## [1.2.0] — 2026-07-20

### Added
* **Developer Hub Page:** Added a permanent, highly detailed, beautifully structured developer page (`/developer`) for **Istiak Rahman** containing interactive digital profiles, support links, and engine details.
* **Command Workstation Shortcuts:** Added professional keyboard listeners:
  - `Ctrl + K` / `Cmd + K`: Toggle Command Palette.
  - `Ctrl + /` / `Cmd + /`: Open Command Palette and focus search query.
  - `Ctrl + Shift + D`: Instantly toggle light & dark modes with active toasts.
  - `Esc`: Instantly escape/close command palettes and modals.
* **Obsidian Sleek Interface Theme:** Customized card glass backdrop, border opacity configurations, and smooth transition animations based on the Sleek reference design.

### Optimized
* Refined and cleaned typography hierarchy across all pages by using tight character spacing (`tracking-tight`) and bold geometric title labels.
* Upgraded search query handling inside command palettes to preserve focus during modal swaps.
